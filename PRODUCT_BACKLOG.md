## Criterio de prioridad

### Alta

Elementos indispensables para que el sistema funcione o para cumplir directamente con la rúbrica.

### Media

Funciones importantes que mejoran la administración, navegación o presentación del sistema.

### Baja

Funciones opcionales que podrían agregarse posteriormente sin afectar el funcionamiento principal.
